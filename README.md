## Repository structure
- `problems/` — Problem statements.
- `solutions/` — Reference solutions by language (e.g. `solutions/java/`).
- `resources/` — Notes and links.

## How to use
1. Clone:
   ```bash
   git clone https://github.com/Chandrasish-Bhattacharya/TCS-Practise-Questions.git
   cd TCS-Practise-Questions
   ```
2. See problems in `problems/` and solutions in `solutions/<language>/`.
3. Run Java solutions (example):
   ```bash
   javac solutions/java/<SolutionFile>.java && java -cp solutions/java Solution < input.txt
   ```

## Contributing
- Fork, create a branch `add/<short-desc>`, add problems or solutions, open a PR.
- Include problem statement, input/output format, and a short explanation with each solution.
